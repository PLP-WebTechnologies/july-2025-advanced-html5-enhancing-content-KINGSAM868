Types of Lists

This section demonstrates various types of lists in HTML5.

Ordered List

These are lists that have a specific order, for example wild animals in a zoo include:

  1. Lions.
  2. Tigers.
  3. Bears.

Unordered List

These are lists that do not have a specific order, for example a shopping list includes:

  • Apples
  • Milk
  • Bread

Definition List

This is a list of key terms and their definition, for example:

HTML5
The fifth and current major version of the HTML standard.
Attribute
A modifier of an HTML element, providing additional information about the element.

Nested List

This is a list that contains other lists, for example the following is a project breakdown:

  • Project Alpha
    1. Planning
      • Define scope
      • Set timeline
    2. Development
    3. Testing
